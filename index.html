Dataset description

The Dataset is found on Kaggle and is open source. There are 3 files in this dataset out of which I choose the Bike Sharing which is very huge but has categorical and numerical columns which can be suitable for visualizations. The visualization plots planduration vs passholdertype. passholdertype is a categorical column and planduration is numerical. To make the plot more interactive I have also added one more categorical column called triproutecategory.
